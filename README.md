# Vincenzo Barager – Portfolio

Live Site: [vincenzo-portfolio.vercel.app](https://vincenzo-portfolio.vercel.app/)

## Overview

This is my personal portfolio website built to showcase my projects, skills, and experience as a Computer Science student. It highlights work across full-stack development, Unity/AR applications, and low-level programming.

## Features

* Project showcase with descriptions and technologies used
* Direct links to GitHub repositories and live demos
* Clean and minimal UI focused on readability
* Responsive design for desktop and mobile

## Tech Stack

* Next.js (framework, handles SSR and routing)
* TypeScript (language used to write app logic)
* Tailwind CSS (styling)
* Vercel (deployment)

## Project Showcase

* **MyMangaReader** – Full-stack manga tracking app with search, list management, and personal recommendations built using Next.js, Node.js, MongoDB
* **FIT AR Navigation App** – Unity + Vuforia mobile AR navigation system
* **Assembly Casino Game** – x86 Assembly game using Irvine32
* **Rhythm Runner** – Unity-based rhythm game

## Running Locally

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## Purpose

This portfolio is continuously updated as I build new projects and improve existing ones. It serves as a central hub for recruiters and collaborators to view my work.
